# 💰 Smart Expense Tracker

A simple command-line based Expense Tracker built using Python to help users manage daily spending.

---

## 📖 Description
The Smart Expense Tracker allows users to record, view, and analyze their expenses efficiently. It stores data using CSV file handling and provides category-wise summaries.

---

## 🚀 Features
- Add daily expenses
- View all recorded expenses
- Category-wise summary
- Persistent storage using CSV file

---

## 🛠️ Tech Stack
- Python
- CSV (File Handling)
